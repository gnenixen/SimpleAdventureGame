Скрипты в этой папке взяты из моих личных проектов, для упрощения стиль кода в них не изменён а некоторые ненужные моменты удалены

Работа с файлами стала менее удобной ввиду невозможности использования плагинов, поэтому все пути до необходимых файлов прописаны в gapi.gd напряму, без возможности всё удобно распределить в отедльном окно редактора, прошу учесть этот момент.
